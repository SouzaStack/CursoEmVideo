O que são Media Queries?

Media Queries são regras condicionais que permitem aplicar estilos CSS específicos com base em características do dispositivo ou do navegador, como:

Sintaxe

A sintaxe básica de uma Media Query é a seguinte:

                
                @media (condição) {
                    /* Estilos CSS */
                }

A condição pode ser uma ou mais características, combinadas com operadores lógicos como and, or e not.

Tipos de Mídia

As Media Queries podem ser direcionadas para diferentes tipos de mídia:

  • all: Todos os tipos de mídia
  • screen: computador, tablets, smartphones, etc.
  • print: Impressoras
  • tv: Televisores
  • projection: Projetores
  • handheld: Dispositivos portáteis (PDAs, etc.)
  • braille: Dispositivos para deficientes visuais
  • embossed: Impressoras braille

Características

As Media Queries podem testar diversas características do dispositivo ou navegador:

  • width: Largura da tela
  • height: Altura da tela
  • device-width: Largura do dispositivo
  • device-height: Altura do dispositivo
  • orientation: Orientação da tela (portrait ou landscape)
  • resolution: Resolução da tela
  • aspect-ratio: Proporção da tela
  • device-aspect-ratio: Proporção do dispositivo
  • color: Número de bits por componente de cor
  • color-index: Número de entradas na tabela de cores
  • monochrome: Dispositivo monocromático
  • scan: Tipo de varredura (interlace ou progressive)
  • grid: Dispositivo em grade (telas de texto)
  • pointer: Tipo de ponteiro (none, coarse, fine)
  • hover: Capacidade de passar o mouse (none, hover)

Exemplos

Exemplo 1: Estilos para telas menores que 768px de largura

                    
                    @media (max-width: 767px) {
                        body {
                            font-size: 14px;
                        }
                        .menu {
                            display: none;
                        }
                    }

Exemplo 2: Estilos para impressão

                    
                    @media print {
                        body {
                            font-size: 12pt;
                        }
                        .no-print {
                            display: none;
                        }
                    }

Vantagens

O uso de Media Queries oferece diversas vantagens:

  • Criação de layouts responsivos
  • Otimização da experiência do usuário em diferentes dispositivos
  • Melhor acessibilidade
  • Maior flexibilidade no design

Conclusão

As Media Queries são uma ferramenta essencial para qualquer desenvolvedor web que deseja criar sites modernos e responsivos. Com elas, é possível adaptar o design para diferentes dispositivos e tamanhos de tela, garantindo que o conteúdo seja exibido da melhor forma possível para cada usuário.
